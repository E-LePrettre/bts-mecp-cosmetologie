# 23 - Objectifs et ressources

Seance FE3 sur la penetration cutanee et l'efficacite des actifs cosmetiques. Les etudiants apprennent les 3 voies de penetration, le role du logP et de la polarite, et les facteurs naturels/galeniques qui modulent la penetration. Coordination PC : polarite, hydrophilie/lipophilie.

---

## Objectifs

### Objectifs scientifiques

- Decrire la **barriere cutanee** (couche cornee, modele briques et mortier)
- Connaitre les **3 voies de penetration** (intercellulaire, transcellulaire, annexielle)
- Comprendre le **logP** et la regle du logP optimal (1-3)
- Identifier les **facteurs naturels** (epaisseur, hydratation, age, zone, T)
- Identifier les **facteurs galeniques** (forme, concentration, promoteurs, occlusion, pH, vecteurs)
- Argumenter le lien **structure -> penetration -> efficacite** (Q11 E2)

## Test rapide : Penetration cutanee

```
1. Quel est le modele de la couche cornee ?
   -> Briques (corneocytes) et mortier (ciment lipidique)

2. Quelle est la voie de penetration principale ?
   -> Voie INTERCELLULAIRE (dans le ciment lipidique, ~95 %)

3. Quel est le logP optimal pour la penetration ?
   -> Entre 1 et 3 (moderement lipophile)

4. Pourquoi l'acide hyaluronique (haut PM) ne penetre-t-il pas ?
   -> Masse >> 500 Da, molecule trop grosse

5. Citez 3 facteurs galeniques favorisant la penetration.
   -> Concentration elevee, promoteurs d'absorption, occlusion (ou pH, vecteurs, forme galenique)

6. Pourquoi une molecule neutre penetre-t-elle mieux qu'une molecule ionisee ?
   -> Le ciment lipidique est apolaire, il repousse les charges
```

---

## Points cles

| Notion | Ce qu'il faut retenir |
|--------|----------------------|
| Couche cornee | Barriere principale : briques (corneocytes) + mortier (ciment lipidique) |
| Voie intercellulaire | 95 % de la penetration, dans le ciment lipidique |
| logP 1-3 | Zone optimale de penetration |
| < 500 Da | Taille maximale pour traverser |
| Promoteurs | Desorganisent le ciment (propylene glycol, ethanol) |
| Occlusion | Film occlusif = hydratation = penetration haute |
| pH | Forme neutre penetre mieux que forme ionisee |
